Straight Lines — Study Notes

1. Vertical and Horizontal Lines ▼

Vertical & Horizontal Lines
Horizontal
Equation: y = c
Gradient = 0
Parallel to x-axis
Vertical
Equation: x = a
Gradient: undefined
Parallel to y-axis

Horizontal Lines

A horizontal line has the equation:

y = c
  • Every point has the same y-coordinate.
  • The gradient is 0 (no vertical change).
  • Example: y = 3 passes through (0,3), (1,3), (5,3).

Vertical Lines

x = a
  • Every point has the same x-coordinate.
  • The gradient is undefined (division by zero).
  • Example: x = −2 passes through (−2,0), (−2,5).
Horizontal = y = c (gradient 0). Vertical = x = a (gradient undefined). Don't mix them up!

2. Points on a Line ▼

Testing if a Point Lies on a Line
1

Take point (x,y)
→
2

Substitute into equation
→
3

LHS = RHS?
If LHS = RHS after substitution, the point is ON the line.

Example

Does (2,7) lie on y = 3x + 1? Sub: 7 = 3(2)+1 = 7. Yes!

Does (1,5) lie on y = 2x + 1? Sub: 5 ≠ 2(1)+1 = 3. No!

This works for any form of equation. Just plug in x and y and check equality.

3. Axes Intercepts ▼

Finding Intercepts
x-intercept
Set y = 0, solve for x
Point: (x, 0)
y-intercept
Set x = 0, solve for y
Point: (0, y)
x-intercept: set y = 0 and solve for x
y-intercept: set x = 0 and solve for y

Gradient from intercepts

Given x-intercept (a,0) and y-intercept (0,b):

m = (b − 0) / (0 − a) = −b/a
x-intercept means y = 0; y-intercept means x = 0. Many students confuse these!

4. Graphing from a Table of Values ▼

Table of Values Method
1

→
2

→
3

→
4

Draw line
  1. Choose at least 3 x-values (e.g. −2, 0, 2).
  2. Substitute each x to find y.
  3. Plot each (x,y) on the grid.
  4. Draw a straight line through all points with a ruler.
Use 3+ points to check your work. If one doesn't line up, you made a calculation error.

5. Gradient-Intercept Form ▼

y = mx + c — The Key Formula
Key Formula
y = mx + c
m = gradient
Rise / Run
m = (y₂−y₁)/(x₂−x₁)
c = y-intercept
Crosses y-axis at (0, c)
m > 0
Line slopes upward /
m < 0
Line slopes downward \
y = mx + c
  • m = gradient (slope): rise over run.
  • c = y-intercept: where line crosses the y-axis.

Drawing using m and c:

  1. Plot (0, c) on the y-axis.
  2. From there, go "rise" up/down and "run" right.
  3. Draw the line through both points.
Gradient from two points: m = (y₂ − y₁) / (x₂ − x₁)
If m = 2, the line rises 2 for every 1 across. If m = −3, it falls 3 for every 1 across.

6. General Form ▼

General Form ↔ Gradient-Intercept
General
ax + by = d
↔
Gradient-Intercept
y = mx + c
ax + by = d

Converting to y = mx + c:

  1. by = −ax + d
  2. y = (−a/b)x + d/b
  3. So m = −a/b and c = d/b

Graphing in general form:

Find the two intercepts and draw through them.

General form avoids fractions. E.g. x + 2y = 6 is neater than y = −0.5x + 3.

7. Finding the Equation of a Line ▼

Three Methods
Method 1
Given m and c
y = mx + c
Method 2
Given m + point (a,b)
y − b = m(x − a)
Method 3
Given two points
Find m first, then Method 2

Method 1: Gradient + y-intercept

y = mx + c

Method 2: Gradient + a point (a,b)

y − b = m(x − a)

Method 3: Two points

  1. Find m = (y₂ − y₁) / (x₂ − x₁)
  2. Use either point with Method 2.
The point-gradient formula y − b = m(x − a) is the most versatile. It works every time!
